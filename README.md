## Para instalar o projeto

```bash
$ npm install
```

## Para compilar e rodar o projeto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# Para ver o Banco de dados

```bash
$ npm run studio
```
