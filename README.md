### Para instalar o backend do projeto

```bash
$ cd projeto-care-back
$ npm install
$ npm run generate
```

### Para compilar e rodar o backend do projeto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Para acessar o Banco de dados

```bash
$ npm run studio
```
